## Events

| Event Key     | Description                                                                                                                                                                            |
| ------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `gsit:sit`    | Makes the player sit.                                                                                                                                                                  |
| `gsit:crawl`  | Makes the player crawl.                                                                                                                                                                |
| `gsit:<POSE>` | Makes the player take a specific pose. You can find the list of available poses [here](https://github.com/gecolay/GSit/blob/main/core/src/main/java/dev/geco/gsit/model/PoseType.java) |
