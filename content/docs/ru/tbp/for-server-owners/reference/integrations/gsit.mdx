## События

| Event Key     | Описание                                                                                                                                            |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `gsit:sit`    | Заставляет сесть.                                                                                                                                   |
| `gsit:crawl`  | Заставляет ползти.                                                                                                                                  |
| `gsit:<POSE>` | Заставляет принять определённую позу. [Список поз](https://github.com/gecolay/GSit/blob/main/core/src/main/java/dev/geco/gsit/model/PoseType.java). |
