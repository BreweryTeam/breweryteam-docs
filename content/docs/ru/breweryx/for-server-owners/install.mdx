## Требования

- **Java 21+**
- Spigot/Paper 1.20.2+

<Callout title="Более старые версии Minecraft/Java" type="warning">

BreweryX требует Java 21. Поскольку версии игры ниже 1.20.5 используют более старую версию Java, BreweryX **не будет** работать на них, если только вы не обновите Java.  
Кроме того, мы больше не поддерживаем более старые версии игры, поэтому вы можете столкнуться с багами на них даже с Java 21. Если вам крайне необходимо использовать старую версию игры, установите более ранний релиз BreweryX, или получите поддержку на нашем <a href={meta.media.discord}>Дискорд-сервере</a>.

</Callout>

## Первая установка

<Steps>

<Step>
    
Загрузите плагин с одного из следующих ресурсов:
    
- [Modrinth](https://modrinth.com/plugin/breweryx) (рекомендовано)
- [Hangar](https://hangar.papermc.io/Jsinco/BreweryX)
- [GitHub релизы](https://github.com/BreweryTeam/BreweryX/releases)
- [Наш Jenkins](https://ci.nadwey.pl/) (дев билды, не рекомендовано)

</Step>

<Step>

Установите плагин:

- Остановите сервер
- Положите JAR-файл в папку `/plugins`
- Запустите сервер

</Step>

<Step>Поздравления, плагин установлен</Step>

</Steps>

## Переход с оригинального Brewery

<Callout title="Миграция с Brewery" type="info">

Как правило, переход с оригинального Brewery на Brewery**X** проходит без проблем. Однако большие изменения в Brewery**X** могут привести к некоторым затруднениям во время перехода.

Поэтому если у вас возникнут проблемы с этим, не стесняйтесь попросить помощи на нашем <a href={meta.media.discord}>Дискорд-сервере</a>.

</Callout>

<Steps>

<Step>

Удалите оригинальный Brewery:

- Остановите сервер
- **Удалите JAR-файл оригинального плагина**
- **НЕ УДАЛЯЙТЕ ПАПКУ** Brewery

</Step>

<Step>

Установите Brewery**X** на сервер:

- Положите JAR-файл Brewery**X** в папку `/plugins`
- Запустите сервер

</Step>

<Step>Brewery**X** должен автоматически перенести данные оригинального плагина в свою папку.</Step>

</Steps>
